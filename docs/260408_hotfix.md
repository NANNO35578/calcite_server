#### 1. summary截断问题
**修改文件：** `calcite/controllers/OcrController.cc`

- **修改** title 格式为 `ocr_YYYYMMDD_HHMMSS`（如 `ocr_20260408_143052`）

#### 2. OCR内容清洗
**新增** `sanitizeText()` 函数，在插入数据库前对 markdown 文本进行清洗：

| 处理项 | 处理方式 |
|--------|----------|
| Null 字节 (`\0`) | 直接移除 |
| 正常空白 (`\n`, `\r`, `\t`) | 保留 |
| 其他控制字符 (`\x01-\x1F`) | 转义为 `\xXX` 格式 |
| UTF-8 多字节序列 | 验证合法性，丢弃无效序列 |
